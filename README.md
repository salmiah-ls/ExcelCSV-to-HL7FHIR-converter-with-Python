# ExcelCSV to HL7FHIR converter with Python

Currently developing a healthcare data engineering project inspired by Indonesia’s SATUSEHAT ecosystem.

The project focuses on transforming raw laboratory data (Excel/CSV) into HL7 FHIR-compliant resources ready for integration into national healthcare platforms.

Main components:
• Python for data processing and transformation
• FHIR Observation resource mapping
• Data validation and standardization
• Configurable mapping for different laboratory formats
• Modular pipeline architecture
• Exploration of AWS-based deployment for scalable processing

This project highlights real-world challenges in healthcare interoperability, including messy data handling, terminology mapping, validation, and scalable pipeline design.

Continued exploration in health informatics, data engineering, and cloud-based healthcare systems remains an exciting journey.
